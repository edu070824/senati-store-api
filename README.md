# SENATI STORE API - Mi proyecto de tienda tecnológica

API para gestionar inventario de una tienda de tecnología. La hice para el curso de SENATI.

## 🎯 Qué hace este proyecto

Es una API que sirve como puente entre el cliente  y la base de datos. Permite crear, leer, actualizar y eliminar productos, categorías y usuarios.

## 🛠️ Tecnologías que usé

| Tecnología | Para qué sirve |
|------------|----------------|
| Node.js    | Para correr JavaScript en el servidor |
| Express.js | Para crear las rutas de la API |
| MongoDB    | Donde se guardan los datos |
| Mongoose   | Para conectar Node con MongoDB |
| JWT        | Para los tokens de autenticación |
| bcryptjs   | Para encriptar contraseñas |
| CORS       | Para que otros dominios puedan acceder |
| dotenv     | Para usar variables de entorno |
| Nodemon    | Reinicia el servidor automáticamente |
| Docker     | Para empaquetar la app |
